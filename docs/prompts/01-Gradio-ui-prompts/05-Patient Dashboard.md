# 05-Patient Dashboard
Create a Patient Dashboard using Gradio.

FEATURES:
1. Book Appointment
2. View Upcoming Appointments
3. View Medical History
4. View Prescriptions
5. Upload Documents
6. Edit Profile

APPOINTMENT BOOKING:
- Fetch available doctors
- Fetch available time slots
- Prevent double booking
- Confirm before submission

UI REQUIREMENTS:
- Calendar style slot selection
- Clear confirmation dialog
- Mobile-friendly

OUTPUT:
Generate full patient dashboard UI with API integration.
